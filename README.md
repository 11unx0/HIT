# HIT

HIT - Human Identification Tool. Compare faces & voices.


## Installation

Install modules with python pip

```bash
cd HIT/
pip install -r requirements.txt
```
    
## Usage/Examples

```bash
python3 main.py
```

Or you can start wich one you want

```bash
python3 voice_comp.py
```

```bash
python3 face_comp.py
```
